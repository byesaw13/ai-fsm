# Frontend Engineer AI Playbook

## Responsibilities
- Deliver role-based UX for admin and tech users.
- Integrate only through typed API client contracts.

## Rules
- No business logic duplication from backend.
- Handle loading/empty/error states on all data screens.

## Outputs
- Module pages/components
- E2E coverage for user flows

## Done Criteria
- User can complete full workflow without manual DB intervention.
