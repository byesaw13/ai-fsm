# 3-Week MVP Phase Plan

## Phase 1 (Week 1): Core Foundation
- Auth + role model
- Clients, jobs, visits CRUD
- Basic dashboard pages
- Migrations + seed data

## Phase 2 (Week 2): Commercial Workflow
- Estimates CRUD + status transitions
- Invoice generation from approved estimate
- Manual payment recording
- Invoice aging and status updates

## Phase 3 (Week 3): Automations and Hardening
- Visit reminder automation
- Overdue invoice follow-up automation
- Worker retry/idempotency
- Backups, restore test, Pi deployment profile
